# CDMSealAPIServer

CDM FHE testing based on [Microsoft SEAL](https://github.com/microsoft/SEAL.git)

## About the project

### Summary
- FHE encrypt the result of CDM request (file base)
- FHE add/sum the encrypted texts
- FHE decrpyt the sum of encrpyted texts

### Function
- FHE encrpyt/decrpyt
- FHE calculate
